5) Objetivo do trabalho?

1. Criação de um banco de dados de dados que derivam de um vídeo que captura pontos de fixação (olhar) da pessoa em ambiente real durante a condução

2. Análise de comportamento visual de condutores de rua para classificar os pontos críticos afim de compreender o peso (importância) dos objetos diante dos cenários, nisso fazer a extração de um relatório individual (privado) mostrando as pontuações em relação as variáveis utilizadas para classificação e comparando com a média da classificacao de outros condutores

6) Qual o produto final? 

Uma classificação para ajudar os condutores de rua a entenderem se estao dirigindo de acordo com outros condutores, afim de evitar um possivel acidente 

7) Quem é o usuário final deste produto?

Condutores de rua

8) O que o usuário recebe de benefício ao usar esse produto? 

Uma análise detalhada de como está a condução dele tentando auxiliar ele a ser um condutor melhor

9) Quais as funcionalidades da ferramenta (visão do usuário)?

Dashboard de análise critica individual, mostrando os dados daquele vídeo
Dashboard de administrador para ver como está o andamento, visibilidade e utilização da plataforma
